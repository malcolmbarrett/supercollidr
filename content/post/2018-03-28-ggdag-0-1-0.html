Iâ€™m pleased to announce the release of ggdag 0.1.0 in CRAN! ggdag uses the powerful dagitty package to create and analyze structural causal models and plot them using ggplot2 and ggraph in a tidy, consistent, and easy manner. You can use dagitty objects directly in ggdag, but ggdag also includes wrappers to make DAGs using a more R-like syntax:

# install.packages("ggdag")
library(ggdag)

dag <- dagify(y ~ x + z,
       x ~ z) %>% 
  tidy_dagitty()
